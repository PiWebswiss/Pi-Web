AI Model to classify fossils

Identifying fossils can be a time-consuming task that relies on expert knowledge of fossil morphology. This task is particularly challenging due to the often fragmented and degraded nature of fossils.

The goal of the model is to provide a system that can accurately identify, classify, and recognize fossils based on their images.

The model was trained on over 80,000 images across 46 different categories. We were able to achieve better results using fewer models. These models are part of my Capstone Project, which you can find on GitHub. Capstone-Project

This model can determine if an image is a fossil or non-fossil with 93% accuracy and has an accuracy of 90% in identifying specific types of fossils. Please note that this model is not 100% accurate and has its limitations. For instance, this model was trained on 46 different categories and therefore can only classify images based on those categories.
